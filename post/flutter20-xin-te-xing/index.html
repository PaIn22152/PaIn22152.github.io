Flutter2.0新特性

2021-03-04

写在前面

时间2021-03-04，Flutter发布2.0版本
更新文档medium.com
接下来聊一聊Flutter2.0的一些新特性。

1.Flutter Web

flutter对web的支持，已经从beta channel到了stable channel，可以直接在stable创建一个web应用。为此flutter做了一些针对web的优化，包括CanvasKit-based renderer和 Link widget等

2.Null Safety

健全的空安全机制，主要得益于dart语言升级到2.12

3.Desktop

对桌面的支持已经加入到stable channel中的早期版本，可以视作“beta snapshot”，最终稳定版本即将到来。想要尝试桌面版本，可以切换到beta channel，或者使用‘flutter config’启用某个桌面配置设置（例如enable-macos-desktop）

4.Flutter Folio

这是一个示例应用，目前flutter支持三个平台（Android，Ios，Web），以及将来会支持的平台（Windows，MacOS，Linux），那么怎么样才能让编写出来的程序适配不同的屏幕，不同的输入模式，不同的使用习惯。Folio回答了这些问题。源码，其他博客

5.Google移动广告插件

之前做过一个flutter项目，需要接入Admob广告和fb广告，因为没有这两个广告平台都没有flutter端的支持，所以只能是在Android端接入Android 广告sdk，Ios端接入Ios 广告sdk，然后使用flutter和原生端通信，而且还不能实现原生广告。现在Google广告提供了对Flutter端的支持，可以直接在flutter使用Google广告，包括插屏，banner，原生，激励等广告。相信不久的将来，fb广告也会提供对flutter的支持。

6.New iOS features

新的CupertinoSearchTextField提供了iOS搜索栏UI。
CupertinoFormSection，CupertinoFormRow和CupertinoTextFormFieldRow部件更容易产生与iOS的分段式视觉美学验证表单字段。
还有一些对于Ios平台性能上的优化

7.New widgets

Autocomplete和ScaffoldMessenger

8.Add-to-App

Add-to-App这个功能，支持在现有的原生应用基础上，增加Flutter Module，然后原生和flutter实例相互跳转

9.其他

Flutter Fix，Flutter DevTools等等，详见以上更新文档。
